Welcome to FoodCheck 🥦!

Explore These Features!

FoodCheck helps users search food products, compare snacks, and save favorite products using nutrition data from the Open Food Facts API.

  • Learn More About the Project
    • Head to the "About Page" to learn more about FoodCheck, the project goals, stakeholders, and tools used.
  • Product Nutrition Lookup
    • Search for a packaged food product by entering a barcode to view nutrition information, calories, sugar, fat, protein, and allergens.
  • Snack Showdown Comparison
    • Compare two snacks side-by-side and see which snack has the stronger nutrition value (based on the Nutri-Score).
  • Save Favorite Products
    • Save products to your favorites list and load them later from the database.
