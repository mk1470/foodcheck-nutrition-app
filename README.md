# Title
FoodCheck: Nutrition Lookup & Snack Showdown

## Description
FoodCheck is a web application that helps users make better food choices by using the Open Food Facts API. Users can search a food product by barcode, view nutrition information, save favorite products, and compare two snacks using the Snack Showdown feature.

## Target Browsers
This application is designed for modern desktop browsers including Google Chrome, Safari, Microsoft Edge, and Firefox. The layout is also responsive for mobile browsers on iOS and Android.
